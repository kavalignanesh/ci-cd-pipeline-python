# CI/CD Pipeline using GitHub Actions 🚀

This project demonstrates a basic CI/CD pipeline that automatically runs a Python script on every code push.

## 🔧 Tech Stack
- Python
- Git
- GitHub Actions

## ⚙️ Features
- Automated workflow triggered on push
- YAML-based CI pipeline
- Executes Python script automatically

## ▶️ How it works
Whenever code is pushed to the repository, GitHub Actions triggers a workflow that runs the Python script.

## 📌 Output
The pipeline runs successfully with a green tick in the Actions tab.
